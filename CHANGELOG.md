# Changelog

All notable changes to **KhataFlow** will be documented in this file.

---

## [1.0.0] - 2026-06-25

### Added
- Fingerprint and 4-digit PIN authentication lock screen.
- Deep Teal-themed layout supporting Material Design 3 guidelines.
- Statement generation exporting beautifully formatted PDF ledgers.
- Push reminder notification scheduler for ledger balances.
- Soft-delete Trash management.

### Fixed
- Addressed duplicate `shareStatementPdf` and missing `crossAxisAlignment` compiler errors.
- Resolved type-lookup issues on constant `Icon` widget parameters.
- Hardened secure storage retrieval on Android.
